# 主题配置目录

本目录用于管理应用的主题样式配置。

## 职责
- 统一管理应用主题
- 提供浅色/深色主题支持
- 实现主题动态切换
- 确保UI风格统一

## 文件说明
- `app_theme.dart`: 主题基础配置
- `color_schemes.dart`: 配色方案
- `text_styles.dart`: 文字样式
- `dimensions.dart`: 尺寸定义
- `decorations.dart`: 装饰样式

## 开发规范
1. 遵循Material 3设计规范
2. 支持iOS原生风格适配
3. 所有样式必须主题化配置
4. 禁止硬编码样式属性
5. 必须支持深色模式
6. 确保无障碍支持

## 主题配置项
- 颜色
- 字体
- 间距
- 圆角
- 阴影
- 动画
- 图标
